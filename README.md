# Симулятор курьера

Небольшая 2D-игра на C# и Windows Forms.

Игрок управляет курьером и должен развезти все заказы по адресам до конца смены.  
У игры есть особенность: не каждый заказ можно доставить в любое время, потому что в игре меняется время суток. Из-за этого приходится смотреть, какой заказ когда доступен, и планировать маршрут по карте.

## Что есть в игре

- перемещение курьера по карте
- 10 домов с получателями
- заказы с разным временем доставки
- смена времени суток
- маршрутный лист
- победа, если все заказы доставлены
- поражение, если закончилось время смены
- стартовый экран
- экран завершения игры
- пауза

## Управление

- `WASD` или стрелки — движение
- `E` — передать заказ
- `Q` — открыть маршрутный лист
- `R` — начать новую смену
- `Esc` — пауза
- `Enter` — начать игру

## Суть игры

Нужно успеть закрыть все заказы до конца смены.  
Если подойти к дому не в то время, заказ отдать не получится. Поэтому игра не только про передвижение, но и про выбор, куда идти прямо сейчас.

## По коду

Проект сделан на Windows Forms.  
Логика, карта, контроллер и отрисовка разделены по MVC, чтобы проект не превращался в один большой файл.

Основные части проекта:

- `GameSession.cs` — заказы, время суток, победа и поражение
- `GameWorld.cs` — состояние игрового мира
- `GameMap.cs` — карта и дома
- `GameController.cs` — управление игрой
- `Form1.cs` — окно и отрисовка
